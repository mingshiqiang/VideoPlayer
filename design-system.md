# VideoPlayer UI 设计系统

## 项目概述
基于 Qt6 Widgets 的 PC 端本地视频播放器界面设计。目标风格：现代、沉浸、简洁、时尚，兼顾深色模式下的视频观看体验与浅色模式下的界面可操作性。

---

## 1. 设计原则

- **沉浸优先**：视频区域占主导，控件在不使用时自然隐去。
- **低对比边界**：使用半透明、细边框和模糊背景，避免强烈色块分散注意力。
- **一致性**：所有图标、按钮、滑块采用统一的几何语言（圆角、2px 描边图标）。
- **即时反馈**：悬停、点击、拖拽均有明确的视觉响应。
- **无障碍**：控件最小 44×44px 触控区域；颜色对比度符合 WCAG AA。

## 5. 组件规范

### 5.1 控制栏（Bottom Control Bar）
- 高度：68px（含内边距）
- 背景：`--bg-panel` +  backdrop-blur(16px)
- 布局：左侧播放控制 | 中部上一首/下一首 | 右侧功能控制
- 底部不显示文件名，文件名仅在顶部标题栏展示
- 默认透明度 0，鼠标移入视频区域 300ms 渐显；3 秒无操作渐隐

### 5.2 播放按钮
- 尺寸：48×48px（主播放） / 40×40px（控制栏播放）
- 形状：圆形
- 图标：2px 描边，居中
- 状态：
  - 默认：`--text-primary`
  - 悬停：`--bg-hover` 背景 + 缩放 1.05
  - 按下：`--bg-active` + 缩放 0.98

### 5.3 进度条（Seek Bar）
- 高度：4px（常态） / 8px（悬停）
- 圆角：2px / 4px
- 颜色：
  - 已播放：`--accent`
  - 未播放：`rgba(255,255,255,0.20)`
  - 缓冲：`rgba(255,255,255,0.35)`
- 滑块：12px 圆形，悬停时显示
- 预览缩略图：进度条上方 8px，圆角 8px，带时间戳

### 5.4 音量滑块
- 宽度：100px
- 高度：4px
- 展开动画：宽度 0→100px，200ms ease-out
- 图标联动：静音时显示斜杠

### 5.5 侧边播放列表
- 宽度：300px（可拖拽调整 260–420px）
- 背景：`--bg-panel`
- 可折叠：收起时 width 0 + opacity 0，250ms cubic-bezier(0.16,1,0.3,1) 过渡
- 头部右侧含关闭按钮（chevron-right 图标，28×28px），点击收起
- 顶部标题栏右侧含切换按钮（侧边栏图标，38×38px），点击展开/收起，激活态高亮
- 列表项：
  - 高度：64px
  - 缩略图：96×54px，圆角 6px
  - 标题：单行省略
  - 时长：右下角时间戳
  - 悬停：背景 `--bg-hover`
  - 当前项：左侧 3px `--accent` 竖线 + 背景 `--bg-hover`

### 5.6 空状态 / 拖放区
- 中央大图标：64px 电影胶片图标
- 提示文字："拖拽视频文件到此处，或点击打开文件"
- 支持格式标签：MP4 / FLV / MKV / AVI / MOV
- 拖拽进入时：边框高亮 `--accent`，背景轻微提亮

### 5.7 顶部标题栏（自定义无边框窗口）
- 高度：40px
- 左侧：logo + 影片名（文件名唯一展示位置）
- 右侧：播放列表切换按钮 | 最小化 | 最大化 | 关闭
- 平时透明，窗口失焦时显示 `--bg-panel`

## 7. 响应式策略

| 窗口宽度 | 行为 |
|----------|------|
| ≥1280px | 默认显示播放列表 |
| 1024–1279px | 播放列表可显示，但默认收起 |
| 768–1023px | 播放列表以抽屉形式从右侧滑入 |
| <768px | 播放列表全屏浮层，简化控制栏 |

## 10. Qt Widgets 实现建议

- 使用 `QMainWindow` 去除默认标题栏（`Qt::FramelessWindowHint`），自定义 `TitleBar` 控件。
- 视频渲染使用 `QVideoWidget`（Qt6 Multimedia）或第三方解码器嵌入 `QWidget`。
- 控制栏使用 `QWidget` + `QHBoxLayout`，设置 `Qt::AlignBottom`。
- 进度条自定义 `QSlider` 样式（QSS）。
- 播放列表使用 `QListWidget` + 自定义 `ItemDelegate`。
- 无边框窗口拖动：重写 `mousePressEvent` / `mouseMoveEvent`。
- 双击标题栏最大化；双击视频区域切换全屏。
